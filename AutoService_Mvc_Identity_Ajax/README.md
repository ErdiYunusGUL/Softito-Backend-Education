# Auto Service (MVC, Identity, AJAX)

## Projenin Amacı
Araç servis ve bakım işlemlerinin yönetildiği bir web uygulamasıdır. Kullanıcı doğrulama ve yetkilendirme işlemleri içerir. Sayfa yenilenmeden asenkron işlemler yapılmasını (AJAX) sağlar.

## Kullanılan Teknolojiler
- ASP.NET Core MVC
- ASP.NET Core Identity (Üyelik Sistemi)
- AJAX & jQuery
- Entity Framework Core

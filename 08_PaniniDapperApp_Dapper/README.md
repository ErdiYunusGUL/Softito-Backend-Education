# Panini World Cup Sticker App (Dapper)

## Projenin Amacı
Dünya Kupası çıkartmalarının (sticker) koleksiyon yönetimini sağlayan bir uygulamadır. Veritabanı işlemleri için ORM olarak Entity Framework yerine mikro-ORM olan Dapper tercih edilerek performans optimizasyonu sağlanmıştır.

## Kullanılan Teknolojiler
- C# / ASP.NET Core Web API
- Dapper (Micro-ORM)
- SQL Server

# Dart League (Code First)

## Projenin Amacı
Dart ligi takımlarını, oyuncularını ve maç skorlarını yönetmek için tasarlanmıştır. Veritabanı tasarımı tamamen kod üzerinden "Code-First" yaklaşımıyla yapılmıştır.

## Kullanılan Teknolojiler
- C# / ASP.NET Core
- Entity Framework Core (Code-First Yaklaşımı)

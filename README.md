# Softito Backend Education Projeleri

Bu depo, Softito Backend Eğitimi kapsamında geliştirilmiş 10 farklı projeyi içermektedir. Her proje, C# ve ASP.NET Core ekosistemindeki farklı teknolojileri ve mimari yaklaşımları öğrenmek amacıyla geliştirilmiştir.

## Projeler
1. AdvancedGameStore
2. AutoService_Mvc_Identity_Ajax
3. CafePOS
4. DartLeague.CodeFirst
5. FlightSearchApp
6. GymApp.NTier
7. LibraryApp
8. PaniniDapperApp

# Softito Backend Education Projeleri

Bu depo, Softito Backend Eğitimi kapsamında geliştirilmiş 10 farklı projeyi içermektedir. Her proje, C# ve ASP.NET Core ekosistemindeki farklı teknolojileri ve mimari yaklaşımları öğrenmek amacıyla geliştirilmiştir.

## Projeler

1. **01_DartLeague_CodeFirst** (CodeFirst)
2. **02_LibraryApp_DbFirst** (DbFirst)
3. **03_GymApp_NTier_CodeFirst** (Katmanlı CodeFirst)
4. **04_CafePOS_RazorPages** (Razor Pages)
5. **05_AutoService_Mvc_Identity_Ajax** (Ajax, JS, jQuery)
6. **06_ScoutingAPI** (API)
7. **07_FlightSearchApp_API_MVC** (API, MVC)
8. **08_PaniniDapperApp_Dapper** (Dapper)
9. **09_SneakerDropPlatform_Dapper_API** (Dapper, API)
10. **10_AdvancedGameStore_Areas** (Areas)

# Gym App (N-Tier Architecture)

## Projenin Amacı
Bir spor salonunun üye kayıtlarını ve antrenman programlarını yönetmek amacıyla geliştirilmiştir. Proje, kurumsal mimari standartlarına uygun olarak Çok Katmanlı Mimari (N-Tier) ile tasarlanmıştır.

## Kullanılan Teknolojiler
- C# / ASP.NET Core
- Çok Katmanlı Mimari (N-Tier Architecture)
- Entity Framework Core
- Repository Pattern
